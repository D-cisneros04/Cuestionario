3 ¿Como se organiza el internet? habla de las direcciones IP y los DNS
    El internet es una red, que ocupa DNS y direcciones ip para regirse, El sistema de nombres de dominio (DNS) es el directorio telefónico de Internet. Las personas acceden a la información en línea a través de nombres de dominio como nytimes.com o espn.com. Los navegadores web interactúan mediante direcciones de Protocolo de Internet (IP). El DNS traduce los nombres de dominio a direcciones IP para que los navegadores puedan cargar los recursos de Internet.

5 Describe los protocolos que existen en internet, en específico HTTP y HTML
